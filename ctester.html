://ACUnitTest.h

Simple C Unit testing frame work that supports Timed, Random, fixed tests. Includes Color macros, Try and catch and a summary screen.

://Header file documentation:

://assert_true:

Functionalty:

Tests a single test case expression, prints the number of the test and a description when failed test. returns a point value ether 1 for Success or 0 for Failure.

://time_assert_true:

Functionalty:

Tests a functions time, and does a try catch to see if it works, then gives half credit for working code, and full credit for code that is below or equal to quota.

://generated_assert_true:

Functionalty:

Tests a single test case expression, prints the number of the test and a description when failed test, and the value generated to test. returns a point value ether 1 for Success or 0 for Failure.
