://DFS & BFS:

DFS: Depth First Search

Depth First Search, starts at the first node and goes through each branch fully before backtracking. I implmement it in my schedule builder, for topological sorting.

BFS: Breath First Search

Breath First Search, starts at the first node and goes through all vertexes in the same depth before going on the next set of vertexes. It usually takes another data structure like stack or a queue to keep track of the child nodes. I implmement it in my social network project, for printing out a whole network.

://Topological Sort:

what is topological sorting:

"Topological sorting for Directed Acyclic Graph (DAG) is a linear ordering of vertices such that for every directed edge u v, vertex u comes before v in the ordering"

This is used for prerequisites for classes, package managers, and getting the order of a directed graph.
