://Schedule Builder part 1:

A implmentaion of BFS, DFS, Topological sorting algorithms to find prerequisites in a directed graph of Fresno state and CSU Monterey Bay Computer Science classes. This is the first part of the project, in C++ to maximize speed in finding prerequisites. Features Object Oriented programming. This is part 1 of many other parts, I have planned. Am thinking for the next part to be able to visualize the graph using D3.js

://Project_description:

This project came with a actual issue, am having with prerequisites.

://The issue:

1) Fresno state has a mess of prerequisites for CSCI classes.

2) This mess of prerequisites would make a prefect directed graph.

3) How do you know if you had all the prerequisites necessary for a class.

3) What is the most common prerequisite?

://The operations this program will preform:

1) Read a file to generate a graph with a adjacency list for each class.

2) Be able to print out the data in a readable format.

3) Check if a class exists.

4) Print the prerequisites for a class.

5) Get the common prerequisites for two classes.

6) Print in Depth first search and breath first search.
