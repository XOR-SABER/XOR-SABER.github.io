A graph theory problem using markov chaining as a solution. Generating lyrics from markov graphs and random generation. As well as using the C++ to create a system to query and look for files. Schedule Builder part 1:

A implmentaion of BFS, DFS, Topological sorting algorithms to find prerequisites in a directed graph of Fresno state and CSU Monterey Bay Computer Science classes. This is the first part of the project, in C++ to maximize speed in finding prerequisites. Features Object Oriented programming.
